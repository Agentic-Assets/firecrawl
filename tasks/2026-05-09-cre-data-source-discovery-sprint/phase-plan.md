# CRE Data-Source Discovery Sprint

## Goal

Run a compact, reproducible discovery sprint for an AI-native CRE prospecting/origination data platform using the local Firecrawl stack.

## Deliverables

- `CRE_DATA_SOURCE_MAP.md`
- `FIRECRAWL_CRE_EXTRACTION_PLAYBOOK.md`
- `SOURCE_TEST_RESULTS.csv`
- `SAMPLE_SCHEMA.json`
- `SAMPLE_EXTRACTS/`
- `BLOCKED_OR_LOW_VALUE_SOURCES.md`
- `NEXT_BUILD_PLAN.md`

## Verification Gate

- Test at least 20 representative sources.
- Cover at least 6 of the requested source categories.
- Save at least 6 compliant sample extracts.
- Record reproducible commands, method settings, failures, and compliance notes.
- Audit every explicit prompt requirement against saved evidence before completion.
