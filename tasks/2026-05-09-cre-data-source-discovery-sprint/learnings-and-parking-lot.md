# Learnings And Parking Lot

- Treat Firecrawl PDF markdown as discovery/metadata evidence. Table-heavy REIT supplements, market reports, rent rolls, and debt schedules need second-pass table QA before downstream numerical decisions.
- Avoid blocked/high-risk listing marketplaces as default ingestion. Prefer public broker pages, public records, issuer disclosures, APIs, licensed data, partnerships, or manual review.
